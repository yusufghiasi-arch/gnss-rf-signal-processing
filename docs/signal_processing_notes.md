# Signal Processing Notes

This document summarizes the signal-processing concepts used in the demos.

## Main Concepts

- IF/IQ samples
- GPS L1 C/A PRN code generation
- correlation
- code delay search
- Doppler search
- delay-Doppler maps
- peak detection
- TDOA
- FDOA

## Portfolio Goal

The goal of this repository is to demonstrate hands-on experience with raw GNSS/RF signal processing workflows, from sampled data to correlation-based observables.
